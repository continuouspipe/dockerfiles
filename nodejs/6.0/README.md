# NodeJS, for SASS and Grunt

```Dockerfile
FROM quay.io/continuouspipe/nodejs:6.0
```

## How to build
```bash
docker build --pull --tag quay.io/continuouspipe/nodejs:6.0 --rm .
docker push
```
